  TROUBLESHOOTING
═════════════════════════════════════════════════════════════

❌ "Cannot find module sqlite3"
✅ npm install in backend/ folder

❌ "Failed to fetch leads"
✅ Check: npm run dev in backend/ is running

❌ "Database locked"
✅ Only one import at a time. Kill process.

❌ "CORS error"
✅ Backend must be running on :3001
✅ Check NEXT_PUBLIC_API_URL in dashboard/.env.local
